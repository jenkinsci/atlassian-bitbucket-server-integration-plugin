When running a job, Jenkins will use these credentials to check out the source code for builds. For Jenkins to do this, it needs credentials with access to the projects and repositories you want it to build from.

To provide Jenkins with credentials you can:

  • choose credentials from the list
  • add credentials as a Username with password (for the password, you can enter a Bitbucket Server password or a Bitbucket Server personal access token)
  • Leave this empty to use default credentials if they exist.
