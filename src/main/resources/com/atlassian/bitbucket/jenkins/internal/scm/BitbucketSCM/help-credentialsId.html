When running a job, Jenkins requires credentials to checkout the source code for builds. To do this, it needs credentials with access to the projects and repositories you want it to build from.

You can provide Jenkins with credentials here by:

  • selecting credentials from the list
  • adding credentials as a Username with password (for the password, you can enter a Bitbucket Server password or a Bitbucket Server personal access token)
  • leaving this empty to use default credentials if they exist

Alternatively, you can provide Jenkins with SSH credentials below. If you do this, Jenkins will use your SSH credentials instead by default.
